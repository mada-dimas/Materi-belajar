Program Tahsin–Tahfidz Al'Qur'an · Tahun Ajaran 2026/2027 · Santri Putra

Tersimpan
⚠️ Mode tanpa penyimpanan bersama. Data yang diketik hanya tersimpan sementara di tampilan ini dan tidak akan terlihat oleh guru lain atau tersimpan setelah ditutup. Ini biasanya terjadi karena file dibuka langsung (diunduh/dobel klik) — buka kembali lewat jendela Artifact di Claude (jangan file yang diunduh) supaya data tersimpan otomatis dan bisa dibagikan ke ~30 pengguna.
Alur setoran (sesuai Pedoman Kurikulum): Murojaah hafalan lama → Binnadhor (baca sambil melihat mushaf, bagian yang akan dihafal) → setor Hafalan Baru / Ziyadah. Tidak ada setoran ziyadah tanpa binnadhor & murojaah terlebih dahulu. Isi kolom sesuai capaian harian Senin–Sabtu (mis. "Al-Baqarah 1‑5", "✓ lancar", atau catatan singkat). Data tersimpan otomatis — titik hijau di kanan atas menandakan data sudah tersimpan.
